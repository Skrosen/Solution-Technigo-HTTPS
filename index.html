HTTPS - HyperText Transfer Protocol Secure

What is HTTPS?

HTTPS stands for HyperText Transfer Protocol Secure. The S in HTTPS ensures the user a secure website where it's safe to exchange personal data. To guarantee that the data is safely transfered with HTTPS the website needs to use a security certificate.

The security is ensured through a SSL och TLS Certificate which is used to authenticate the identity of the website, without any of these certificates the website tranfers data with HTTP and therefore unencrypted wich puts the users personal information at risk. SSL (Secure Sockets Layer) and TLS (Transport Layer Security) uses public key encryption to ensure security and authenticate the server, client and at the same time encrypts the data.

What websites uses HTTPS?

Most websties uses HTTPS by default, regardless if sensitive data might be exchanged or not. Google is actually flagging websites without SSL or TLS protection as not secure and is also penalizing those websites.
